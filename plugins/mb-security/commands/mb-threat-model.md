# /mb-threat-model

Invoque a skill `mb-threat-modeler`. Mapeie sistema, fronteiras, ativos, atores. Percorra STRIDE + ameaças cripto-específicas (front-running, replay, reorg, drenagem, oráculo, smart contract).

Gere `docs/specs/_active/<feature>/THREAT-MODEL.md`.

Ameaças com severidade CRÍTICA exigem mitigação implementada antes de SHIP. Riscos residuais aceitos devem ser documentados explicitamente.

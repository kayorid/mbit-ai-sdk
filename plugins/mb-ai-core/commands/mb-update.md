# /mb-update

```bash
bash "${CLAUDE_PLUGIN_ROOT}/scripts/update.sh"
```

Verifica versão local vs remoto, exibe changelog, aplica atualização. Major version bump exige confirmação explícita por possíveis breaking changes.

Após atualização, reinicie a sessão do Claude Code.

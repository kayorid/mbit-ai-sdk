# /mb-dashboard

Execute o dashboard ASCII:

```bash
bash "${CLAUDE_PLUGIN_ROOT}/scripts/dashboard.sh"
```

Painel mostra: squad, trimestre, specs ativas, ciclos por trimestre (sparkline), hooks bloqueantes da semana, aprovações, exceções, barra de maturidade, achievements desbloqueados, e próxima ação contextual sugerida.

Use:
- Início do dia para ver onde o squad está.
- Antes de retros para visualizar trimestre.
- Em demos/apresentações para mostrar métricas instantâneas.

# Release Notes

## 1.0.0 — 2026-05-02

Primeira versão do plugin, extraída do projeto wis-lms após validação em produção com a spec de tokenização do design system (4 PRs, ~280 hardcodes migrados).

### Incluído

- **SKILL.md**: instrução completa da skill com fases, artefatos, triggers e anti-patterns
- **Templates**: requirements, design, tasks, status, ADR, retrospective, INDEX
- **Exemplos**: spec de exemplo com todos os artefatos preenchidos
- **Referências**: notação EARS, phase playbook, status tracking, AI harness patterns, integration guide
- **Scripts**: init_spec.sh, update_index.py, validate_spec.py
- **Evals**: casos de teste para validação da skill
